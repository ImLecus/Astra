Astra CSS

Make it easier

Get started

Astra is a CSS framework that makes easier and faster to add styles to your projects. The objective with this framework is to have a solid CSS source with prefabricated components and the capacity to create unique styles with it.

To use this framework, you can put the CDN link as the example below or download it here. 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/ImLecus/Astra@0.2.1/dist/astra.min.css" integrity="sha384-7t7Vz/ci5s9DieF2UNOyTASLyDsuX5ordXN5rGWLapTaop27/WUkiEVbYJ0WB6MA" crossorigin="anonymous">

Animation library

<div class="fade-up">
    <div class="rotate-left"></div>
    <div class="fade-right"></div>
</div>

A website without movement can be boring, isn't it? Well, Astra has the solution for that problem. It comes with an animation library ready to give life to your projects.

Made in Sass

Do you want to customize Astra? Do it. It's as simple as modify the SCSS variables in the main code and change Astra to adapt it to your style. 

$primary-color: #ad842d;

$colors:(
    "sky": #00c0fa,
    "zinc": #565656
);
