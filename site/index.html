Astra

The CSS framework that makes easier and faster to write CSS.

Don't you want to use CDN links? Download source code

Quick and easy to use

Astra was created for both experienced creators and people without much knowledge of CSS. Therefore, its structure is intuitive and quick to use and learn.

<div class="display-3 semiBold grey">
    Astra
</div>
<p class="lead marginY-3">
    Make it easier
</p>
                    
body{
    background-color: var(--astra-grey-100);
    color: var(--astra-grey-800);
    font-family: var(--astra-body-font);
}
Fully customizable

Unleash your most creative side and made beautifull websites with Astra and its CSS variables.

Endless possibilities

With the new function classes and media classes the customization possibilities are infinite!
