Flex

Learn how to create a flex with Astra CSS.


To add a flex in your project you need to use the flex class. It will be, by default, with the row direction, so you can add the column, reverse or column-reverse classes to change the flex direction.

Flex wrap

By default, the flex-wrap property is enabled in the flex. To disable it, use the no-wrap class.
