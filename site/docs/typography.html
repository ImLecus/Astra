Typography

Learn how to write rich text with Astra.


Headings

Usually, we use the h1 to h6 tags to create headings with different size and importance. But maybe you want an element with the size of a h5 but without leaving the properties of the tag. For that problem, Astra has the h1 to h6 classes that you can add to your texts to add the properties of a heading.


This is a h1

This is a div with the h1 class

As you can see, the font size can be changed by adding this classes.


Lead parapraphs

Some parapgraphs are more important than others, and that creates the need of having a new type of paragraph. Astra has the solution with the lead paragraphs. Here is an example:

Hey! I'm an important paragraph.

And this is the content. The details are here.

To use lead paragraphs in your project, use the lead class.


Bold, italic, light and semibold text

To add bold or italic text you can put a b or i tag. But you can also use the b, bold or i classes to make your text bold or italic. Additionally, Astra has the light text, which is used by the light class; and the semibold text, used by the semi-bold class.
