Headers

Learn how to use the Astra features on your custom headers


Creating a header

The first step to create a header in Astra is to apply the .header class on the HTML header.

<header class="header">

<header>

Our header will need to have flex properties. In this example it will have a flex with row direction. Learn more about flex here.

<header class="header flex">

<header>
