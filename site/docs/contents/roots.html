Roots

Learn about the root element and how it works


What are roots?

The root elements are the most basic element container in Astra. They have different sizes and they mark the width of the elements inside them. For example, the .root element has a maximum width of the 75% of the parent element, in this case the body.


Different sizes of roots

Roots have 7 different sizes, here is a table with all of them and their width based on the screen width:

Extra Small (<600px)Small (<768px)Medium (<992px)Large (<1200px)Extra Large (<1440px)Ultra Large (≥1440px)
.root75%75%75%75%75%75%
.rootXS100%600px600px600px600px600px
.rootS100%100%768px768px768px768px
.rootM100%100%100%992px992px992px
.rootL100%100%100%100%1200px1200px
.rootXL100%100%100%100%100%1440px
.rootFull100%100%100%100%100%100%

Creating a root

To create a root element you need to apply one of the previous root classes inside an element, preferably inside a div tag, like the example of below:

<div class="root">
    <p>This is a paragraph inside a root</p>
</div>

Full and default roots

There are two special types of roots: .root and .rootFull. They doesn't have a specified width, because they works with percentages. .rootFull will expand to the entire screen width, and .root will expand only to the 75% of the screen. 

<div class="root">
    <p>This will expand until the 75%</p>
</div>
<div class="rootFull">
    <p>This will expand until the 100%</p>
</div>
