Tables

Learn how to use custom tables in your project

Creating custom tables

To use predefined Astra styles in your tables, you need to add the table class. After that, there are some classes that changes the style of your tables. You can also combine them to make your own table style.

Example: 

<table class="table"> <!-- Content --> </table>
# A B
1 1A 1B
2 2A 2B

Striped tables

The first custom style is the striped table style. To add custom styles, make sure that you have the table class on your table element and add the striped class too, like the example below: 

<table class="table striped"> <!-- Content --> </table>
# A B
1 1A 1B
2 2A 2B

Bordered tables

To add borders to all the elements of your table, you need to add the border class: 

<table class="table border"> <!-- Content --> </table>
# A B
1 1A 1B
2 2A 2B 
<table class="table striped border"> <!-- Content --> </table>
# A B
1 1A 1B
2 2A 2B

Dark tables

To activate the dark mode on your custom tables, you can add the dark class: 

<table class="table border dark"> <!-- Content --> </table>
# A B
1 1A 1B
2 2A 2B 
<table class="table striped border dark"> <!-- Content --> </table>
# A B
1 1A 1B
2 2A 2B
