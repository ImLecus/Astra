Tables

Learn how to change the style of the tables with Astra.


Default tables

By default, tables in Astra looks like this:


A C
1 1A 1B
2 2A 2B
3 3A 3B

To change the style of the tables, Astra has some classes:

Dark tables

You can put a dark table by adding the dark class in the table tag:


A C
1 1A 1B
2 2A 2B
3 3A 3B

Tables without borders

To disable the borders in the table element, add the no-border class:


A C
1 1A 1B
2 2A 2B
3 3A 3B

You can also combine this option with the dark table:


A C
1 1A 1B
2 2A 2B
3 3A 3B

Framework compatibility

If you are working with two or more frameworks at once, use the table class on the table tag for better compatibility. By default, the style will be applied to the table tag without any extra classes.
