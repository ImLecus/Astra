Position

Learn how to position elements with Astra.


Relative

To make an element with relative position use the relative class. This option is enabled by default.


relative
relative
relative

Absolute

To make an element with absolute position use the absolute class and the positioning classes, that are the direction and a number from 0 to 5:


absolute left-5
absolute right-5



Sticky

To make an element sticky you can use the sticky, sticky-top and sticky-bottom classes.To change the position you can use the positioning classes.

Fixed

To make an element to be fixed use the fixed class. To change the position you can use the positioning classes.


Z-Index

To change the z-index property of an element use the z-1 to z-5 classes.


Overflow

You can change the overflow of the elements by using the overflow-x, overflow-y, overflow and no-overflow classes.
