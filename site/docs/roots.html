Roots

Learn the basics of the containers in Astra CSS.


What are roots?


Basically, roots are containers with limited size. They are very useful at responsive design due to its limits are the breakpoints between different device sizes. For example, a XS root will have a maximum space similar to the width of a mobile screen.


Basic root

Additionally, there is a special type of root called basic root. This root will expand always to the 80% of the screen, leaving a margin of 10% to each side. This root is not used to responsive websites, instead it puts a margin to limit the width of the content. To use it, add a div with the root class.
